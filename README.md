Setup
---

```
npm install
```

Add sounds to the `/public/sounds` directory

Run
---

```
npm start
```

or watch files

```
npm run watch
```

API
---

#### GET /

View main page. Play sounds or text-to-speech.

#### GET /sounds/:sound

Play a sound

#### POST /speech

* Body
  * message - string to convert to audio

Text-to-speech

#### POST /slack

* Body
  * text - name of the sound file to play

Play a sound via POST
